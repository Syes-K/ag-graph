快速使用 

// 初始化
var agGraph = new AgGraph({container: "#graph-container"});
// 添加节点
var node1 = agGraph.addNode({
  id: "n1",
  x: -125,
  y: -66,
  size: 40,
  image: "./assets/img/server.svg",
  text: "示例"
});
var node2 = agGraph.addNode({
  id: "n2",
  x: 86,
  y: 53,
  size: 40,
  image: "./assets/img/pc.svg",
  badge: "99"
});
// 添加连线
var line = agGraph.addLine({
  id: "edge1",
  source: 'n1',
  target: 'n2',
  "text": "线段",
  animate: true,
  "pointsData": [
    {x: -121, y: -30},
    {x: -72, y: 36},
    {x: 11, y: -20},
    {x: 61, y: 49}
  ]
});
// 添加运动路径
agGraph.addPath({
  id: "p1",
  source: "n2",
  target: "n1",
  repeat: true
});
ag-graph是什么？
ag-graph是一款专业的图形化界面开发工具集.伴随着大数据, 人工智能的快速发展, 数据可视化走上舞台. ag-graph应运而生.
为什么需要ag-graph？
你是否遇到过需要可视化展示的时候，苦苦寻觅找不到?
或者找到了无法深度定制? 或者憋闷过UI粗糙不够美观?
或者提交issues无人理会叫天不应的抓狂?
ag-graph: “这都不是事儿”.
ag-graph特点和优势
●商业版本, 快速响应特殊需求.
●API简单易用, 成熟稳定, 可扩展性强.
●基于SVG样式易于修改, 想象无极限.
●调试方便, 右击直接查看生成图形.
