ag-graph容器的高度控件是否会进行控制？目前无样式div容器高度较小，没有默认height:100% ?
是的, 需要自己指定宽高, ag-graph灵活的地方也是在这里, 因为天然支持样式来控制宽高, 所以支持所有计算单位(px, vh, %, em, rem, calc等) 

  /* 仅供参考 */
  #ag-graph-container {
    width: 100%;
    min-width: 20vh;
    max-width: 1920px;
    height: calc(100% - 20px);
    min-height: 20em;
    max-height: 50rem;
  }
按照官方API文档操作，添加完连线节点之后，所有元素均不能移动或者操作，是需要继续什么配置吗？为什么不是默认可以操作？
需要启动编辑模式后, 可以进行拖拽和编辑连线等操作, 关闭编辑模式, 可以添加路径轨迹动画. 

  // 初始化
  var agGraph = new AgGraph({container: "#ag-graph-container"});
  // 切换编辑模式
  agGraph.startEdit();
因为默认情况多数以观看展示为主, 编辑操作仅开始配置时需要, 其余时间修改较少, 也有一部分原因是权限控制的需要, 不是所有人都有编辑权限的.
